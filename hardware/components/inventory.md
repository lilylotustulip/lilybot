# Project Inventory 

This is the exact list of hardware components, power supplies, and wires used to build this bot.

## Main Components

| Component Name | Purpose | Key Specifications |
| :--- | :--- | :--- |
| **Raspberry Pi Zero 2 W** | The main brain of the bot | Quad-core CPU, 512MB RAM, Wi-Fi/Bluetooth |
| **MAX98357A I2S Audio Amplifier** | Powers the speaker for audio output | Mono Class D amplifier, I2S interface |
| **INMP441 I2S Omnidirectional Microphone** | Captures voice input for the bot | Digital I2S interface (*Note: Had to buy twice! because i made some erros and ruined it, for more info visit the journal*) |
| **28318R3 Electronic Speaker** | Plays the bot's voice responses | 8 Ohm |
| **Samsung MicroSD Card** | Holds the Operating System and code | 16GB, Class 10 speed |
| **Breadboard** | Used to prototype and map out the circuit connections | Half-size or Full-size (*Note: the breadboard has only 6 wires plugged there so u dont need a big one*) |

## Power & Cabling

| Item | Purpose | Specs |
| :--- | :--- | :--- |
| **Samsung EP-TA200 Adaptive Fast Charger** | Power brick for the Raspberry Pi | 5V / 2A Output |
| **Micro USB Cable** | Delivers power and data from the brick to the Pi | High-quality power/data cable |
| **Female-to-Female Jumper Wires** | Used to connect components together | Main wiring choice used in build |
| **Female-to-Male Jumper Wires** | Used for specific pin-to-board extensions | Used in build |
| **Male-to-Male Jumper Wires** | Leftover wires | *Bought but not used in final assembly* |
